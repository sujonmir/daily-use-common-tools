Open Google Sheets

1 Create a new Google Sheet

  1. Go to sheets.new to create a new blank Google Sheet.
  2. Rename it to something like Freelancer Workspace so you can find it easily.
  3. Don't add any columns or sheets manually — the script will create everything for you (Projects, TimeEntries, Payments, Invoices).

2 Open Apps Script editor

  1. In your new Google Sheet, click Extensions → Apps Script.
  2. Apps Script will open in a new tab with a default Code.gs file.
  3. Delete everything inside Code.gs.

3 Paste the script

  1. Open the script file from this repository:
    📄 rec/freelancer-workspace-appscript.gs
  2. Copy the entire file contents.
  3. Paste it into Code.gs in the Apps Script editor.
  4. Press Ctrl+S (or ⌘+S) to save. Name the project anything you like.

4 Deploy as Web App

  1. Click the blue Deploy button (top-right) → New deployment.
  2. Click the gear icon → choose type Web app.
  3. Settings:
    • Execute as: Me
    • Who has access: Anyone (required so the page can call it)
  4. Click Deploy. You'll be asked to authorize the script — accept all permissions.
  5. Copy the Web app URL shown in the success dialog.
The URL ends in /exec. If you ever change the script and want to redeploy, use Deploy → Manage deployments → Edit (pencil) and change the version to New version. The URL stays the same.

5 Paste the URL above & start

  1. Paste the URL into the field at the top of this page.
  2. Click Save, then Test to verify connection.
  3. Once it shows Connected, head back to the workspace and start adding projects.

Tips & data safety

  • Your data stays in your sheet. Nothing is sent to any third party — the page talks directly to your own Apps Script endpoint.
  • Keep the Apps Script URL private. Anyone with the URL can read/write to your sheet.
  • You can manually edit rows in the Google Sheet — it'll sync on the next page load.
  • Active timer state is stored locally per-browser, so you can stop a timer from the same browser that started it.

Danger zone

Disconnect removes the saved URL from this browser. Your Google Sheet data is not affected.
