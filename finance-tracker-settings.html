Google Sheets Setup Guide

        One-time
      

      

        

          

            
1

            

              Create a new Google Sheet
              
Go to sheets.google.com and create a new spreadsheet.

            

          

          

            
2

            

              Open Apps Script
              
In the spreadsheet go to Extensions → Apps Script. Delete all existing code.

            

          

          

            
3

            

              Paste & deploy the script
              
Copy the script above and paste it in. Then click Deploy → New deployment → Type: Web App → Execute as: Me → Access: Anyone. Copy the deployment URL.

              
⚠ Already deployed before? Go to Deploy → Manage deployments → pencil icon → Version: New version → Deploy.

            

          

          

            
4

            

              Connect below
              
Paste the deployment URL in the Connection section below, enter your secret key to verify, then click Connect.

Backup & Restore

      

      

        

          Backup Now pushes all local transactions to Google Sheets (replaces Sheets data).

          Restore Now pulls all transactions from Google Sheets (replaces local data).

          Your secret key is also your app login key.
